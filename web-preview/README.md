# AgentPad Web Preview

This folder is a lightweight browser preview for quickly checking the v0.2.3 Agent UI direction on a desktop machine.

It is not the Android app runtime. The production app still lives in `android-app` and must be verified on Android, either with a device or an emulator.

Open `index.html` in a browser to review:

- GPT-style Agent chat shell
- Chat and Settings
- in-chat authorized file/image access
- provider presets
- narrow-screen bottom navigation
