# AgentPad v0.2.1-alpha.1

This signed pre-release redesigns AgentPad around real local task threads.

## Included

- Multiple task threads with immutable multi-turn history.
- New follow-ups supersede unexecuted plans and invalidate in-memory approvals.
- Room v1-to-v2 migration preserving old goals, plans, results, and audit events.
- Interrupted-state recovery after process death or device restart.
- Explicit context-compression confirmation with retained original history.
- Attachment metadata in planning context without silently uploading file contents.
- Compact, medium, and expanded layouts without nested vertical lazy lists.
- DeepSeek and custom OpenAI-compatible configuration with a persistent model badge.
- Light and dark themes, plus optional screenshot privacy mode.
- Local redacted crash diagnostics and user-initiated export.
- Debug artifacts named with a short Git SHA; this Release contains only the formally signed APK.

## Upgrade

`AgentPad-v0.2.1-alpha.1.apk` uses the same formal signing identity as v0.2.0 and is intended to install as an in-place upgrade while retaining local data.

## Not included

- AccessibilityService or cross-app automation.
- Shizuku.
- Python, Git, shell, or a Runtime APK.
- Web or desktop clients.
- Silent installation, payment automation, root, or private-data access.

Android will show its normal installation confirmation. Verify the accompanying SHA256 and signing certificate record before installing.
