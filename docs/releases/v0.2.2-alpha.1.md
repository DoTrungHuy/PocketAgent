# AgentPad v0.2.2-alpha.1

This alpha focuses on simplifying the Android native workspace experience.

## Included

- Codex/Zed-inspired workspace with three primary areas: Chat, Tasks, and
  Settings.
- Message bubbles, a fixed composer, and inline task approval cards.
- Provider presets for mainland-friendly OpenAI-compatible services.
- Clearer provider failure categories for rate limits, timeouts, retryable
  service failures, rejected requests, and invalid responses.
- A lightweight desktop browser preview under `web-preview/` for reviewing the
  simplified UI direction without installing an emulator.
- Fixed the manual Android instrumentation workflow path.

## Not included yet

- Real cross-app Accessibility observation, click, input, swipe, or screenshot
  upload tools.
- Shell, Python, Git, or standalone Runtime APK execution.
- Streaming UI, attachment center, capability catalog, and device automation
  preview panels.
- Silent installation, payment automation, password/OTP handling, or lock-screen
  bypass.

This release keeps the same Android security boundary: the model proposes plans,
but local code owns tools, risk normalization, approvals, and auditing.
