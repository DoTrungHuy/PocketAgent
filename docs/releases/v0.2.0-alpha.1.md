# PocketAgent v0.2.0-alpha.1

This is the first signed native Android alpha.

## Included

- Native Kotlin and Jetpack Compose workspace for phones and tablets.
- DeepSeek and custom OpenAI-compatible model configuration.
- Android Keystore protection for API keys.
- Task planning, local risk normalization, parameter-bound approvals, and
  single-use approval tokens.
- User-authorized text document reading and model-assisted summaries.
- Room task history, audit summaries, and DataStore settings.
- Android 16 target SDK and Android 9 minimum SDK.

## Not included yet

- Accessibility-based cross-app automation.
- Shizuku integration.
- Python, Git, or shell Runtime APK.
- Silent installation or background privilege escalation.

Android will show its normal installation confirmation. Keep the APK,
SHA256 file, and signing certificate record together when verifying a download.
