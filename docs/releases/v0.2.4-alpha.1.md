# PocketAgent v0.2.4-alpha.1

This alpha redesigns PocketAgent around permission-gated phone document search.

## Highlights

- Mobile-first chat UI with phone history drawer and wide-screen history sidebar.
- Agent detects document-search intent and asks for Android file/folder authorization.
- Local document indexing for authorized ranges.
- Content search across txt, md, json, xml, html, docx, and text-layer pdf files.
- Model explanation of likely matches with file names, reasons, and snippets.

## Privacy boundary

PocketAgent does not silently scan the whole phone. It only searches files and folders authorized through Android system pickers.

## Compatibility

The app label and release assets are PocketAgent. The Android package id remains `com.agentpad.app` so earlier signed installs can upgrade in place.
