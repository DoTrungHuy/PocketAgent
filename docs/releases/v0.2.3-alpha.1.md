# PocketAgent v0.2.3-alpha.1

This alpha repositions PocketAgent as a mobile phone-content agent with a chat entry point.

## Included

- Chat-first shell with new chat on launch and persistent history in the sidebar.
- Primary surfaces reduced to Chat and Settings.
- Direct agent replies without forcing every message through the visible task-plan workflow.
- In-chat access button for granting files and images to the agent.
- Automatic reading of authorized text files and images when the agent replies.
- Optional vision endpoint/model settings for image-capable OpenAI-compatible providers.
- Browser preview updated to show the agent permission model.

## Boundaries

- File and image access still requires Android user-granted URI permissions.
- Text files are read up to the local size cap; images are sent as data URIs up to the image cap.
- Unsupported binary formats are passed as metadata only until dedicated parsers are added.
- No cross-app Accessibility clicking, typing, payment automation, password/OTP handling, or lock-screen bypass.
