# PocketAgent Roadmap

## v0.2.4-alpha.1

- Rename user-visible product, docs, preview, release assets, and main source files to PocketAgent.
- Keep `com.agentpad.app` and `agentpad.db` for Android upgrade compatibility.
- Replace workflow-first UI with a mobile chat surface and left history drawer.
- Add permission-gated phone document search by content.
- Add local indexing for authorized files/folders.
- Support text extraction for txt, md, json, xml, html, docx, and text-layer pdf files.

## Next

- OCR for scanned PDFs and screenshots.
- Better incremental indexing for large folders.
- Optional media-library search with explicit Android media permissions.
- Stronger result explainability and per-file summaries.
- Background indexing only after a separate user-facing privacy review.
